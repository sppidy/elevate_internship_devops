# DevOps Internship Task 1

## Objective
CI/CD pipeline using GitHub Actions to automate testing, building, and deploying a Node.js app to DockerHub.

## Tech Stack
- GitHub Actions
- Node.js
- Docker
- DockerHub

## Pipeline Flow
1. Code push to `main`
2. Run tests
3. Build Docker image
4. Push to DockerHub
